App Icon 生成器

📱 Android App Icon 尺寸規格：

  • ldpi: 36 x 36px
  • mdpi: 48 x 48px
  • hdpi: 72 x 72px
  • xhdpi: 96 x 96px
  • xxhdpi: 144 x 144px
  • xxxhdpi: 192 x 192px

📁 檔案放置位置：

plambeads/ └── res/ └── icon/ └── android/ ├── ldpi.png ├── mdpi.png ├── hdpi.png ├── xhdpi.png ├── xxhdpi.png └── xxxhdpi.png

🚀 使用步驟：

  1. 點擊每個尺寸的「下載」按鈕
  2. 將下載的 PNG 檔案放入對應的資料夾
  3. 執行 cordova build android
  4. 安裝 APK 即可看到新的圖示！
