Android Splash Screen 生成器

📱 Android Splash Screen 規格：

  • 尺寸： 1080 x 1920px (9:16 比例)
  • 格式： PNG
  • 背景色： 深紫色 (#4a148c)
  • 內容： 您的 logo 居中顯示

📁 檔案放置位置：

plambeads/ └── res/ └── screen/ └── android/ └── splash.png

🚀 使用步驟：

  1. 點擊「下載 Splash Screen」按鈕
  2. 將下載的檔案命名為 splash.png
  3. 放入 plambeads/res/screen/android/ 目錄
  4. 執行 cordova build android
  5. 安裝 APK 即可看到新的啟動畫面！

💡 注意事項：

  • 這個 Splash Screen 只會在應用程式啟動時短暫顯示
  • 之後會自動切換到我們的 React LoadingScreen 動畫
  • 確保目錄結構正確，檔案名稱必須是 splash.png
